1) 정규표현식

- [0-3] : 0에서 3까지 하나 이상

- [0-35-9] : 4를 제외한 숫자범위0에서 3까지, 5에서 9까지 중 하나

- ^ : 맨 앞의 글자 -> 문자열의 시작

- $ : 맨 뒤의 글자 -> 문자열의 끝

- ^(안)[0-9] : 안으로 무조건 시작하고 0-9 사이 숫자 하나

- . : 어떤 글자든 1개 이상 들어가야 함

- [a-zA-Z] : a-z, A-Z 문자열 1개

- [가-힣] : 한글 아무거나 1개

- [가-힣]{3} : 한글 아무거나 3개

- [가-힣]{2,4} : 한글 아무거나 2개~4개

- [a-z]{2,5}@[a-z]{2,5}(.)[a-z] : 소문자 a-z 까지 2글자 이상 5글자 미만, @을 포함하고, a-z 까지 2글자 이상 5글자 미만으로 .


2) 전화번호

- \d : =[0-9]과 같다

- \d{3}-\d{4}-\d{4} : 숫자 000-0000-0000

- \d{3}-\d{3,4}-\d{4} : 숫자 000-000(0000)-0000

- 01[06-9]-\d{3,4}-\d{4} :숫자 01(06~9) -000~0000 - 0000

01[06-9]\d{3,4}\d{4} 혹은 d{3,8} : -을 제외하고 전화번호 표기


3) + 와 * 과 .

- ac+b : c가 1개 이상 포함되어야 한다.

- ac*b : c가 0개 이상 포함되어야 한다.(있어도 그만 없어도 그만)

- \. : 진짜 .
