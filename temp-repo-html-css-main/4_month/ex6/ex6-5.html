5. 플렉스 박스(align-items: center )



* 메모

- 태그#아이디명: 아디가 적힌 태그 생성

- crtl+shift+방향키(↑or↓): 코드 자리 옮기기

- justify-content: 주축을 정렬한다.

  • flex-start: 시작 부분에 정렬
  • flex-end: 끝에 정렬
  • center: 중앙정렬
  • space-between: 요소가 축을 따라 펼쳐짐
  • space-around: 요소가 축을 따라 펼쳐지지만 가장자리 주변에도 공간이 있음

-align-items: 교차축을 정렬한다.

  • stretch: 요소의 길이와 교차축의 길이를 같게함
  • flex-start: 시작 위치로 정렬
  • flex-end: 끝 위치로 정렬
  • center: 중앙으로 정렬
