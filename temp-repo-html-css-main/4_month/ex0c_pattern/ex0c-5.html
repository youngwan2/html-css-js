* 메모

- * : 전체 선택자(순위가 가장 낮다). 만약에 * 보다 높은 선택자인 div 를 사용하면 * 에 적용된 효과는 사라지고, div 태그에 적용된 효과로 적용된다. 여기서 태그를 선택하는 것을 '유형 선택자'라고 한다.

- : (클론), ; (세미클론), . (닷) ;

- () : 소괄호

- {} : 중괄호

- [] : 대괄호

- em : body 태그에서 font-size 기본값인 16px을 기준으로 보겠다(상대 사이즈).

- rem : html 태그에서 font-size 기본값인 16px을 기준으로 보겠다(root 상대 사이즈).

- text-aligin : 텍스트를 정렬한다(left, center, right 등).

- font-size 폰트 크기를 지정한다.

- css는 따로 파일로 만들어서 관리하는 것이 유지보수, 웹 페이지 성능, 패키지 만들기 등에 유용하다.

- .클래스명 : 클래스 선택자

- #아이디명 : 아이디 선택자
