flex : 축약형(glow:늘어뜨림, shrink, basis)

flex-grow:늘린다.

flex-shrink: flex 아이템의 크기가 flex 컨테이너의 크기 보다 클 경우 아이템이 얼마나 축소되어야 하는지 지정한다. 2로 지정하면 초과하는 크기의 2배를 줄인다는 의미이다.

flex-basis: flex 아이템의 초기 크기를 지정할 때 사용한다. 200px 로 지정하면 아이템 초기 크기가 200px로 시작한다.
